Required Fields

Field Name Type Required Description
email String Required User email address
password String Required User password (8+ chars, uppercase, lowercase, number)
first_name String Required User first name
last_name String Required User last name
phone_number String Required User phone number (E.164 format)
tenant_id String Required Tenant/Store ID
date_of_birth String (ISO) Required Date of birth (for age verification)
accept_terms Boolean Required Accept terms and conditions
accept_privacy Boolean Required Accept privacy policy

Optional Fields

Field Name Type Default Description
accept_marketing Boolean false Accept marketing communications
referral_code String null Referral code (if applicable)
enable_voice_auth Boolean false Enable voice authentication
voice_audio File (UploadFile) null Voice sample for authentication
device_id String null Device identifier
role String "Customer" User role
registration_source String "web" Registration source (mobile/web/kiosk)
